enum UnityEngine.Timeline.TimelineClip.ClipExtrapolation
strong

How the clip handles time outside its start and end range.

Enumerator
None 

No extrapolation is applied.

Hold 

Hold the time at the end value of the clip.

Loop 

Repeat time values outside the start/end range.

PingPong 

Repeat time values outside the start/end range, reversing direction at each loop

Continue 

Time values are passed in without modification, extending beyond the clips range

◆ CreateCurves()

void UnityEngine.Timeline.TimelineClip.CreateCurves ( string  curvesClipName)
inline

Creates an AnimationClip to store animated properties for the attached PlayableAsset.

If curves already exists for this clip, this method produces no result regardless of the value specified for curvesClipName.

When used from the editor, this method attempts to save the created curves clip to the TimelineAsset. The TimelineAsset must already exist in the AssetDatabase to save the curves clip. If the TimelineAsset does not exist, the curves clip is still created but it is not saved.

Parameters
curvesClipNameThe name of the AnimationClip to create. This method does not ensure unique names. If you want a unique clip name, you must provide one. See ObjectNames.GetUniqueName for information on a method that creates unique names.

Implements UnityEngine.Timeline.ICurvesOwner.
