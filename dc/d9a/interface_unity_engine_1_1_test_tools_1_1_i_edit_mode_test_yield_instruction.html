Detailed Description

In an Edit Mode test, you can use IEditModeTestYieldInstruction interface to implement your own instruction. There are also a couple of commonly used implementations available:

  • EnterPlayMore
  • ExitPlayMode
  • RecompileScripts
  • WaitForDomainReload
